REFRAIN — QUOTES CURATION INSTRUCTIONS
=======================================
Refrain displays time- and season-matched verses from folk song collections.
This file is the single source of truth for curation state, conventions, and
collection notes. Read it at the start of every session.

─────────────────────────────────────────────────────────────────────────────
THE LIBRARY FILES
─────────────────────────────────────────────────────────────────────────────
Each collection has its own JSON file. At session end, updated files are
generated as downloadable outputs; the user replaces the project files manually.

─────────────────────────────────────────────────────────────────────────────
LYRICS FILE ARCHITECTURE
─────────────────────────────────────────────────────────────────────────────
Lyrics are stored in per-collection JSON files. At deploy time,
generate_lyrics.py merges all files and writes src/lyrics.js; generate_quotes.py
does the same for src/quotes.js. To add a new collection, create the JSON file
and add it to LYRICS_FILES / QUOTE_FILES in the relevant script.

── Campbell & Sharp (collection: "campbell-sharp") ──────────────────────────
Source: English Folk Songs from the Southern Appalachians, Olive Dame Campbell
and Cecil J. Sharp (G.P. Putnam's Sons, New York, 1917). 122 songs.
Text sources: Archive.org scans (englishfolksongs00camp_0 is higher quality).
Secondary: mainlynorfolk.info, bluegrassmessengers.com.
Do not conflate with the 1932 Karpeles edition.
Cite as: Song Title (Campbell/Sharp 1917, No. N)
Key format: campbellsharpNN (e.g. campbellsharp76)
Status: COMPLETE. 21 approved entries; full 122-song collection assessed.
Breakdown: morning 10, night 10, evening 3, afternoon 0;
spring 2, summer 2, winter 2, no season 15.
Afternoon is genuinely absent — the repertoire does not carry it.
Verified source notes:
- "I dreamt last night" (Early, Early in the Spring) — NOT in 1917 edition.
- "Tonight or in the morning soon" (Wife of Usher's Well) — NOT in 1917 edition.
- Lady Isabel (No. 2), version A: "three hours before it was day" (st. 10) —
  caught on full pass, not keyword search.
- Grey Cock (Mrs. Jane Gentry version): confirmed in 1917 edition (evening, summer).

── John A. Lomax — Cowboy Songs (collection: "lomax") ───────────────────────
Source: Cowboy Songs and Other Frontier Ballads, John A. Lomax
(Sturgis & Walton, New York, 1910). ~150 songs.
Text sources: Project Gutenberg (gutenberg.org/files/21300/21300-8.txt).
Authorship: mostly traditional/anonymous. Known authored songs in the file:
- Harry Stephens — Night-Herding Song (c. 1890)
- Brewster Higley — Home on the Range (words, 1873)

── John A. and Alan Lomax — American Ballads and Folk Songs (collection: "lomax-1934")
Source: American Ballads and Folk Songs, John A. Lomax and Alan Lomax
(Macmillan, New York, 1934). ~200 songs.
Text sources: Archive.org OCR is degraded — verify against physical pages.
The Lomaxes composited stanzas from multiple sources; treat texts as
representative rather than definitive.
Important: some songs are reprinted from other collections. Always check the
Lomax footnote before assigning attribution (e.g. Every Night When the Sun Goes
In is from Sharp/Karpeles 1932, not original Lomax material).
Cite as: Song Title (Lomax 1934)
Key format: lomax1934SongTitle in camelCase
Status: COMPLETE. 1 approved entry (Gerry's Rocks — morning).
Verified: Once More a-Lumbering Go is not in this collection (it's from Folk
Song U.S.A., 1947). Low Bridge (Erie Canal) is a 1905 composition, not traditional.

── Sharp/Karpeles — Southern Appalachians 1932 (collection: "karpeles-appalachian")
Source: English Folk Songs from the Southern Appalachians, revised and enlarged
edition edited by Maud Karpeles (Oxford University Press, 1932). 2 vols, ~274 songs.
Text sources: mainlynorfolk.info (primary); Archive.org / Loomis House Press reprint.
Cross-check all candidates against refrain-quotes-campbell-sharp.json.
Note: Published 1932, pre-1928 US cutoff — public domain in the US. Karpeles's
editorial contributions are not yet public domain in the UK, but verse texts are safe.
Cite as: Song Title (Sharp/Karpeles 1932, Vol. N, No. N)
Key format: karpelesappalachianSongTitle in camelCase
Status: COMPLETE. 9 approved entries; all 72 Vol. I ballads and Vol. II assessed.
Breakdown: morning 6, night 3, evening 1; spring 1, no season 8. No afternoon.
Verified source notes:
- Every Night When the Sun Goes In: reprinted from Sharp/Karpeles 1932, not Lomax.
- The Chickens They Are Crowing: from Ben Finlay, Clay Co., KY, 1917 (play-party).
- Bright Morning Star: hymns section, new to 1932 edition; treat as traditional.
- Little Musgrave: Version A from Mrs. Becky Griffin, Big Laurel, NC, 1916.
  Not in Campbell-Sharp file despite being in 1917 edition — cite to 1932.
- Fair Margaret and Sweet William: opens 'Sweet William rose one morning bright'
  (not 'one May morning') — no spring tag.
- Cruel Ship's Carpenter: 'Early one morning before the break of day' confirmed
  in Sharp R (Clinton Fitzgerald, Afton, VA, 1918).

── Sabine Baring-Gould — Songs of the West (collection: "baring-gould") ─────
Source: Songs of the West, S. Baring-Gould, H. Fleetwood Sheppard, and F.W.

Text sources: Project Gutenberg (gutenberg.org/ebooks/56625) is the primary

Note: Baring-Gould expurgated and rewrote some texts. His manuscripts are at

Cite as: Song Title (Songs of the West, No. N)
Key format: baringgouldN (e.g. baringgould89)
Status: COMPLETE. 29 approved entries; full 121-song collection triaged via

Breakdown: morning 17, night 8, evening 4, afternoon 1; spring 8, summer 4,
winter 2, no season 14. No autumn entries.
Verified: Upon a Sunday Morning (No. 3) — melody is not a genuine folk tune,
but text is collected from Robert Hard, South Brent; included.
The Hal-an-Tow (No. 24) — May 8th spring ceremony; tagged spring, not summer.

── Lucy Broadwood — English Traditional Songs and Carols (collection: "broadwood")
Source: English Traditional Songs and Carols, Lucy E. Broadwood
(Boosey & Co., London, 1908). 38 songs.
Text sources: the 1908 edition (Archive.org, identifier: englishtraditio00broagoog).
PDF scan has no extractable text layer; OCR via pytesseract at 180–200 DPI gives
good results. Full-collection keyword triage is feasible via rasterised OCR.
Note: A substantial number of songs came from Henry Burstow, a bellringer and
cobbler from Horsham, Sussex. Several carols came from the Goby family, gypsies
in Sussex and Surrey.
Important: The "rambling all this night" stanza of The Moon Shines Bright appears
in Broadwood's 1893 English County Songs (Thomas Gray, Weston, Hertfordshire),
NOT in the 1908 collection. The 1908 Goby version has six theological stanzas only.
Do not conflate the two publications.
Cite as: Song Title (Broadwood, English Traditional Songs and Carols)
Key format: broadwoodSongTitle in camelCase (e.g. broadwoodMoonShinesBright)
Status: COMPLETE. 5 approved entries; full 38-song collection assessed.
Breakdown: morning 2, evening 1, night 2; spring 2, summer 1, no season 2.

── W. Roy Mackenzie — Ballads and Sea Songs from Nova Scotia (collection: "mackenzie")
Source: Ballads and Sea Songs from Nova Scotia, W. Roy Mackenzie
(Harvard University Press, Cambridge, 1928). 162 songs.
Text sources: Archive.org scan (balladsseasongsf00mack). The PDF has a text layer
on prose pages but not reliably on verse pages; pdftotext extracts introduction
text cleanly. Verse pages were verified visually via pdftoppm at 180 DPI.
Songs collected primarily from Pictou and Colchester counties, Nova Scotia,
c. 1908–1912. Two-thirds of the songs are from River John, Pictou County.
Copyright: Published 1928 Harvard University Press. Copyright was not renewed
(pre-1964 publication); public domain in the US. Safe to use verse texts.
Note: Mackenzie's internal song numbering differs from the TOC by one in places
(TOC No. 26 = internal No. 27, Young Edmund). Always cite from the internal number
as printed in the book.
Note: Gerry's Rocks appears in two versions (A and B), both with "Sunday morning"
opener — duplicate of Lomax 1934 entry, not added.
Note: The OCR text layer is unreliable for verse stanzas; always verify stanza
text against the rasterised scan before finalising any entry.
Cite as: Song Title (Mackenzie 1928, No. N)
Key format: mackenzieTitle in camelCase (e.g. mackenzieDawningOfTheDay)
Status: COMPLETE. 9 approved entries; full 162-song collection triaged.
Breakdown: morning 4, evening 3, night 2; summer 3, spring 1, winter 1, no season 4.
No afternoon entries. Evening/night coverage notably improved by this collection.
Approved entries:
- No. 56 The Dawning of the Day — morning, summer (milkmaid at dawn, June)
- No. 54 The Chippewa Stream — evening, summer (washing linens, roses in bloom)
- No. 64 The Dark-eyed Sailor — evening (night is coming and the day is gone)
- No. 61 Mary of the Wild Moor — night, winter (bitter winds, wild moor)
- No. 71 The Banks of Brandywine — morning, spring (moon in trees, early May)
- No. 66 Janie on the Moor — morning (seaside, hills in flower)
- No. 72 The Bonny Bunch of Roses — morning, summer (ocean border, June birdsong)
- No. 27 Young Edmund — night (public house by the shore)
- No. 70 The Banks of Claudie — evening/night (dark and stormy night, love at sea)

─────────────────────────────────────────────────────────────────────────────
CURATION RULES
─────────────────────────────────────────────────────────────────────────────
A quote is eligible if it:
- Contains an explicit time-of-day marker (morning, afternoon, evening, night).
  Strong incidental imagery is acceptable (firelight/candlelight for night,
  dew/milkmaid for morning) but birdsong alone is not a time marker.
- Works as a standalone image or moment. Plot-dependence is no longer a strict
  barrier since full lyrics are available — the test is whether the image lands.
  Floating pronouns that are genuinely opaque mid-narrative remain a reason to
  reject.
- Has sufficient lyrical quality to stand alone.

Additional conventions:
- Light modernisation of spelling is acceptable. Scots vocabulary is preserved
  where it adds atmosphere (lang, mirk, een, waly, linn, tirled at the pin, etc.)
  — flag unusual words for discussion.
- No invented time or season tags. Season tags only where textually supported.
- Riddle and impossible-task ballads are not categorically excluded.
- Always show full quote text when presenting candidates in a table.
- Keep sessions collection-specific. One session = one collection.
- Process ~5 songs per batch; present only genuine candidates with brief reasoning.

─────────────────────────────────────────────────────────────────────────────
DEPLOYING TO THE APP
─────────────────────────────────────────────────────────────────────────────
Run all commands from the repo root (~/refrain/):

  cp ~/Downloads/refrain-quotes-[collection].json scripts/
  cp ~/Downloads/refrain-lyrics-[collection].json scripts/
  cp ~/Downloads/generate_quotes.py scripts/      # if updated
  cp ~/Downloads/generate_lyrics.py scripts/      # if updated
  cp ~/Downloads/AboutPage.jsx src/               # if updated
  cp ~/Downloads/PrivacyPage.jsx src/             # if updated
  cp ~/Downloads/AboutView.swift ios/Refrain/Refrain/  # if updated
  python3 scripts/generate_quotes.py
  python3 scripts/generate_lyrics.py
  # Verify DEV_MODE = false in src/config.js
  git add . && git commit -m "..." && git push

Vercel auto-deploys on push to main. generate_quotes.py skips hold,
pending-wordsmith, and retired entries automatically.
If the Vercel webhook fails: git commit --allow-empty -m "Trigger redeploy" && git push

iOS content updates (quote additions) require a new App Store build and
resubmission, since quotes.json is bundled. Web app updates instantly.

⚠️  ANDROID content updates also require a new Play Store build and resubmission.
After running the generate scripts, copy JSON files to Android assets:
  cp quotes.json android/app/src/main/assets/
  cp lyrics.json android/app/src/main/assets/
Then build a new signed AAB and submit via Play Store Console.

  web fetching when a clean PDF is available. mainlynorfolk.info is reliable

CODE ARCHITECTURE

  MainActivity.kt          — entry point, widget deep link handling

  ui/LyricsSheet.kt        — full-height ModalBottomSheet with stanza scroll

  ui/widget/RefrainWidget.kt         — Glance home screen widget

Minimum SDK: 26 (Android 8.0). Target SDK: 35.
Pool filtering: quotes with empty season array match any season (same as iOS/web).
